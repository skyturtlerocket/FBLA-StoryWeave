How To Use StoryWeave

Choose a Story

On the navigation bar located at the top, there's an Explore button. From here, you can choose a story you want to explore through!
You can view the descriptions by hovering over each option on the Explore page.

Using a story

Each story allows a user to choose their own way through the story, by giving them options for them to choose. These will take the user through different outcomes in the story!
Each story allows the user to stop interacting with it by pressing the STOP button.
A user can skip an animation by clicking the Skip Animation button if you don't want to go through it.
Once you complete a story, a report will generate summarizing what happened and the choices you made.

Additional Questions

If you have any additional questions or need more help, refer to our interactive Q&A help menu!

Using the Interactive Q&A

In the interactive Q&A, you will see various questions. If you click on a question, it will display it in a text box with the question and answer.
The help menu is designed so that you can't choose a question more than once, however, clearing the text box will reset the questions you've clicked on.
